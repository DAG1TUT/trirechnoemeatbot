# Деплой бота на Railway

## Переменные окружения в Railway

В настройках сервиса Railway (Variables) добавь:

| Переменная    | Описание                          | Пример              |
|---------------|-----------------------------------|---------------------|
| `BOT_TOKEN`   | Токен бота от @BotFather          | `123:AAH...`        |
| `ADMIN_IDS`   | Telegram ID руководителя (отчёты) | `8194223689`        |

Опционально:

- `DATABASE_URL` — по умолчанию используется SQLite в контейнере (данные сбрасываются при перезапуске). Для постоянной БД можно подключить PostgreSQL в Railway и указать его URL.

## После деплоя

1. Открой бота в Telegram и отправь `/start`.
2. С аккаунта из `ADMIN_IDS` отправь боту `/start` — откроется меню руководителя, на этот чат будут приходить отчёты.
